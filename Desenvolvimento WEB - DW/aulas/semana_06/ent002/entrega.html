O que é HTML?

O HTML, também conhecido como hiper text markup language (linguagem de marcação de hiper texto), é um bloco de notas criado por Tim Berners Lee, para desenvolver anotações científicas e enviar em forma de mensagem.

Com o passa do tempo, foram surgindo novas versões do html, que são:

O HTML 2.0 (1995), HTML 3.2 (1997), HTML 4.01 (1999) e o XHTML 1.0 (2000), até chegar na versão que é usado hoje em dia, que é o html 5.
