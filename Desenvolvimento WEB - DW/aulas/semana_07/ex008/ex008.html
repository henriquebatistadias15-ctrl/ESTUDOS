Formatação de textos

Negrito / Destaque

Na frase, tenho um termo em negrito usando a tag <b> (sem sentido semântico)

Nesta frasen tenho um termo importante usando a tag <strong> (com força semântica)

Itálico / Ênfase

Nesta frase, tenho um termo em itálico usando a tag <I> (sem sentido semântico)

Nesta frase, termos um termo em ênfase usando a tag <EM> (com ênfase semântica)

Terto marcado

Podemos criar também um texto marcado usando a tag <MARK>

Texto pequeno

Podemos também colocar textos reduzidos usando a tag <SMALL>

Texto deletado

Podemos também marcar um texto como excluído usando a tag <DEL>

Texto inserido

Um texto inserido ganha ênfaseusando a tag <INS>

Texto sobrescrito

Um texto sobrescrito permite escrever coisas como X 2 + 2 usando a tag <SUP>

Texto subscrito

Também podemos usar texto subscrito para fazer H2O usando <SUB>
